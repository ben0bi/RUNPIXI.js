All written by hand, just that you know it.

Fast introduction:
* Create a div with an id (here: pixiscreen) with a given (fixed) width and height,
* include the source files with the ben0biJSLoader (github) :) or directly in the html file:

** pixi.js from pixijs.com,
** then RUNPIXI.js or RUNPIXI_B.js,

* create a loop function or use NUFF for your game logic (in a <script>):

* then call PIXI.RUN("pixiscreen", loop); and look what the console gives you.

* after that, call PIXI.PIXELATED(); for a pixel-perfect view for your pixel-perfect game. :)

More docs later, see README.txt for further information about version A.
